## Claude

計測点（T3の位置）は 外側の凸角（くさびの段差が下がる瞬間の角）として自動検出されています。

  具体的には 44〜52行目のロジックです。

  for i in range(len(z_surface) - 1):
      if depth_vals[i+1] < depth_vals[i]:   # 深さが減少 = 段差の角
          corner_z.append(int(z_surface[i+1]))
          corner_x.append(int(x_surface[i]))

  手順のまとめ：

  1. 対象範囲：z_rec_start ～ z_rec_end（探触子が乗るピッチ1個分）を z_surface として定義（30〜34行目）
  2. 表面プロファイル生成：kusabi_depth(z) で各 z の深さを計算。くさび形状なので z が進むにつれ深さが線形に減少し、ピッチ境界でリセット（36〜39行目）
  3. 凸角の検出：隣接する z で depth_vals[i+1] < depth_vals[i] （深さが急に減少＝段差）になる点が計測点。グリッド離散化により斜面が階段状になるため、各段の「外側の角」が T3 の計測点になる
  4. 計測点数：step_size = 1（1メッシュ刻み）なので、段差は depth / step_size = mn_d 個

  T1・T5 の計測点指定はこのファイルには含まれておらず、kusabi_surface_sim_*.py 側で別途定義されています。
